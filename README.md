Contact Book API (FastAPI)
Пет-проект для отработки навыков построения REST API и работы с асинхронным Python. Основной фокус здесь на бэкенде, архитектуре БД и правильной упаковке приложения в Docker.
Интерфейс (фронт часть) - это простая статическая страница на JS, чтобы визуализировать работу API.

Backend:
1. FastAPI: Используются асинхронные роуты для высокой производительности.
2. SQLAlchemy: Работа с базой через ORM, логика моделей вынесена в отдельные слои.
3. Pydantic: Строгая валидация входящих данных (схемы в schemas.py).
4. Uvicorn: ASGI-сервер для запуска приложения.

Инфраструктура:
Проект полностью контейнеризирован. В docker-compose.yml описана связка:
1. API Service: Python контейнер с бэкендом.
2. Web Service: Легкий Nginx для отдачи фронтенда.

Как запустить (Docker):
docker-compose up --build

После запуска бэкенд со всей документацией доступен тут: http://localhost:8000/docs (Swagger UI). Это лучший способ проверить все методы API.

Структура кода:
1. BACKEND/api.py - эндпоинты и бизнес-логика.
2. BACKEND/database.py - настройка подключения к БД и описание таблиц.
3. BACKEND/schemas.py - схемы данных для валидации (Pydantic).
4. FRONTEND/ - просто обертка для взаимодействия с API.

Планы по Python-части:
1. Покрыть API тестами (Pytest).
2. Добавить интеграцию с Redis для кэширования.
3. Реализовать миграции БД через Alembic.
